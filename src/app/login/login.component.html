Welcome to the ATM Project

My goal with this project is to create a fully functional ATM system that simulates basic transactions like withdrawals, deposits, and balance inquiries.

For this project, I'll be using Angular as the front-end framework and integrating it with a backend hard-coded server to handle the ATM operations.

The resources required for this project include:

  • A development environment with Node.js and Angular CLI installed
  • A Bootstrap CSS library for styling
  • Angular Material UI library for a better user interface
  • Database (front-end hard-coded) to store account balances and transaction records

Some of the functions that I plan to implement include:

  • User authentication and login
  • Displaying account balance and transaction history
  • Withdrawal amount selection with appropriate validations
  • Deposit function to add funds to the account
  • ATM transaction logic to handle balance updates
